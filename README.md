# 坦克大战1.0(TankGame)

## 项目介绍

## 各文件说明
Bomb.java   爆炸效果类
EnemyTank.java   敌方坦克类
MyJrame.java   主窗口，运行其中的main方法即可运行程序
MyPanel.java   绘制游戏区域
MyTank.java   我方坦克类
Node.java   记录退出游戏时剩余敌方坦克的坐标方向
Recorder.java   记录退出游戏时的得分以及其他信息，并写入到myRecord.txt文件
Shot.java   射击类
Tank.java   坦克类，是MyTank.java和EnemyTank.java的父类
